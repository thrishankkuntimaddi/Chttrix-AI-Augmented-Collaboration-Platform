# Chttrix Desktop

> Electron shell that wraps the Chttrix web application — zero changes to `client/` or `server/`.

## Architecture

```
Electron Main Process (Node.js)
  └── BrowserWindow
        └── Loads: https://app.chttrix.com  (or localhost:3000 in dev)
              └── Renderer = the existing React web app
```

The web app runs identically to the browser version. Electron adds:
- **OS Native Notifications** — shown when the window is not focused
- **System Tray** — unread badge count, mute toggle, quick access
- **Deep Links** — `chttrix://` protocol opens the app and navigates to context
- **Auto-Updates** — electron-updater checks GitHub Releases on startup
- **Badge Count** — dock (macOS) / taskbar (Windows) unread count


## How to Run (Development)

**Prerequisites**: The Chttrix web app must be running at `http://localhost:3000`.

```bash
# 1. Install dependencies
cd desktop
npm install

# 2. Start (web app must be running separately)
npm run dev
```

Or from the repo root (starts web + Electron together):
```bash
# Terminal 1: start the web client
npm run client

# Terminal 2: start Electron pointing at localhost
npm run desktop
```

## How to Build (Production)

```bash
# From desktop/
npm run build       # All platforms
npm run build:mac   # macOS only
npm run build:win   # Windows only
npm run build:linux # Linux only

# Built packages appear in desktop/dist/
```

## Web App Integration (Optional)

The web app can detect it is running inside Electron and opt into native features:

```js
// In any React component or hook:
const isDesktop = Boolean(window.chttrixDesktop);

// Set unread badge count on app icon
window.chttrixDesktop?.setBadgeCount(unreadCount);

// Show an OS notification
window.chttrixDesktop?.showNotification({
  title: "New message from Alice",
  body: "Hey, are you free for a call?",
  conversationId: "dm-abc123",  // optional — click navigates here
});

// Listen for deep links
const unsub = window.chttrixDesktop?.on("deep-link", (url) => {
  // e.g. "chttrix://dm/user456" — parse and navigate
});
// cleanup:
unsub?.();

// Show update banner when available
window.chttrixDesktop?.on("update-downloaded", ({ version }) => {
  // show in-app prompt: "Version X.X.X is ready. Restart to update."
});
```

## Security Notes

| Setting | Value | Rationale |
|---|---|---|
| `nodeIntegration` | `false` | Renderer has no direct Node access |
| `contextIsolation` | `true` | Preload isolated from renderer world |
| `sandbox` | `true` | Renderer runs in OS sandbox |
| `webSecurity` | `true` | Same-origin enforced |
| IPC channels | Allow-listed | Prevents arbitrary channel access |
| External URLs | `shell.openExternal` | Never loaded in Electron window |

## Auto-Update Configuration

Update the `publish` section in `electron.config.js` to point at your release host.
For GitHub Releases:

```js
publish: [{ provider: "github", owner: "chttrix", repo: "chttrix-desktop-releases" }]
```

A `latest.yml` file is auto-generated by electron-builder alongside each release artifact.
